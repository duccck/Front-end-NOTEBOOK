## 判断二进制位是否为1
1. 检查第i位时是否为1，可以让n与 2^i进行与&运算，当且仅当n的第i位为1时，运算结果不为 0

    (n & (1 << i)) != 0 ? 1 : 0;

2.  n & (n - 1)，直到结果为0，运算了多少次，即有多少个1

|运算|规律|
|--|--|
|1 & 0 = 0|运算1次为0，有1个1|
|10 & 1 = 0|运算1次为0，有1个1|
|11 & 10 = 10; 10 & 1 = 0|运算2次为0，有2个1|

    var hammingWeight = function(n) {
    let count = 0;
    
    // 方法一
    // for(let i = 0; i < 32; i++) {
    //     if((n & (1 << i)) != 0) count++;
    // }
    
    //方法二
    while(n) {
        n &= n - 1;
        count++;
    }
    return count;
    };
