## 递归
把一个问题逐级（栈的形式）分解成一组子问题，子问题与原问题是纵向、同类的关系
**总体思想**

 - 问题：输入变量 x 的函数 f(x)
 - 寻找 g()，使得 f(x) = g(f(x-1)) (即 f(x) -> f(x-1))
 - 已知f(0)，通过 f(0) 和 g() 求出 f(x) 的值

**三个要点**

 1. 递归式，将原问题划分为子问题
 2. 递归出口，最小问题的求解
 3. 界函数，问题规模变化的函数，保证递归规模向出口条件靠拢

*注：如果回溯过程中存在体积大的变量（如数组），递归层次很深的情况下，可能会出现栈溢出。可以使用全局变量或动态数组解决*
